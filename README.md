# 🏠 House Price Analysis Dashboard

This is an interactive Streamlit dashboard built using the Kaggle House Prices dataset.

## 🚀 Features
- Interactive filters (Quality & Price Range)
- Key metrics (Total Houses, Average Price, Median Price)
- Sale price distribution visualization
- Living area vs Sale Price analysis
- Neighborhood price comparison

## 🛠 Tech Stack
- Python
- Streamlit
- Pandas
- Matplotlib
- Seaborn

## ▶ Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
